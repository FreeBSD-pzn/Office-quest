                      Главное окно                            
Используя  клавиши-стрелки,  подведите  указатель  под  букву,
с  которой  начинается  фамилия  абонента.    Нажмите   ENTER.
Если фамилия записана латинскими буквами,  нажмите клавишу F9.
Если Вы забыли фамилию абонента,для поиска нажмите клавишу F6.
Для  регистрации  нового  абонента  используйте  клавишу   F5.
Нажав  клавишу  F7,  Вы  получите  доступ   к   справочнику  с
телефонами  городов.    ESC  -  выход  в  ДЕЛОВОЙ   КАЛЕHДАРЬ.
                  Справочник по городам                       
Для  быстрого  поиска  нужного  Вам города начинайте вводить с
клавиатуры его название. Если очередная буква не  отображается
в окне ПОИСК, значит этого города в справочнике нет.Для поиска
также  пользуйтесь  клавишами - стрелками, PgUp, PgDn.        
Для редактирования записей  в  справочнике нажмите клавишу F4.
Для   добавления   нового   города   используйте  клавишу  F5.
               ESC - выход из справочника.                    
                  Окно "Карточка абонента"                    
Перед   Вами   карточка  с  информацией о выбранном  абоненте.
Для  редактирования   информации   используйте   клавишу   F4.
Есть  возможность смотреть и  редактировать  окно   примечаний
( если  Вам  известен  пароль )  -  используйте  клавишу   F2.
Для  того,  чтобы  воспользоваться   услугами   справочника  с
кодами и справ. телефонами городов  СHГ  нажмите  клавишу  F7.
ESC  -  выход  в   предыдущее   окно  ( " список абонентов" ).
                      Окно "КАЛЕНДАРЬ".  F9-печать дел,встpеч.
   Используя клавиши-стрелки, Вы  можете  выбрать  нужный день
месяца.  Для получения доступа  к записям выбранного дня нужно
нажать на клавишу ENTER.  Перейти в другой месяц - PgDn, PgUp.
Быстро   вернуться   в   сегодняшний   день  -  клавиша   F10.
Перейти  в  другие  окна ( "Встречи", "Дела" ) - клавиша  TAB.
Откpыть  БЛОКHОТ - F8.  Листать:  клавиши-стpелки (дата),  F8.
Войти в ТЕЛЕФОHHЫЙ СПРАВОЧHИК-F3.    ESC - выход из программы.
                       Окно "ВСТРЕЧИ".   F9-печать дел,встpеч.
  Используя клавиши-стрелки, Вы можете перемещаться по записям
в окне. Корректировать запись - F4.  Если при этом Вы изменили
время встречи - записи автоматически перепишутся в  хронологи-
ческом порядке.  В окне "Встречи за день" гистограмма наглядно
покажет  распределение  запланированных  встреч в течение дня.
Добавить встречу - клавиша F5. Добавить дело - F6. TAB-перейти
в другое окно. DEL,INS-стереть записи. ESC-выход из программы.
                        Окно "ДЕЛА".     F9-печать дел,встpеч.
Используя клавиши-стрелки,  Вы  можете перемещаться по записям
в  окне.   Корректировать  запись - F4.   Добавить  дело - F6.
Добавить  встречу - клавиша  F5.    DEL,INS - стереть  записи.
Отметить выполненное дело  -  клавиша ENTER  ( слева от записи
появится галочка).  УЧТИТЕ!  Если  Вы  не  отметите выполнение
какого-нибудь дела,на следующий день оно опять появится вновь.
TAB - перейти  в  другое  окно.   ESC  -  выход  из программы.
             Главное окно со списком абонентов.               
Горизонтальные клавиши-стрелки для выбора другой буквы алфави-
та.  Вертикальные клавиши-стрелки  для  перемещения  по списку
абонентов.  Для быстрого поиска фамилии абонента в списке наж-
мите клавишу F2. Далее начинайте набирать по одной букве фами-
лию. Если фамилия абонента присутствует в списке, маркер уста-
новится на ней.  Просмотр карточки абонента - F3. Гоpода - F7.
Добавить абонета-F5.Удалить абонента-INS.Выход из ТЕЛ.СПР-ESC.
